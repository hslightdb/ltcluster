2.2. Installing ltcluster from packages

2.2.1. RedHat/CentOS/Fedora
2.2.2. Debian/Ubuntu

We recommend installing ltcluster using the available packages for your system.

2.2.1. RedHat/CentOS/Fedora

ltcluster RPM packages for RedHat/CentOS variants and Fedora are available from the Hundsun LightDB public repository; see following section for details.

Note

Currently the Hundsun LightDB public repository provides support for RedHat/CentOS versions 5, 6 and 7. Support for version 8 is available via the PGDG repository; see below for details.

RPM packages for ltcluster are also available via Yum through the LightDB Global Development Group (PGDG) RPM repository (https://yum.postgresql.org/). Follow the instructions for your distribution (RedHat, CentOS, Fedora, etc.) and architecture as detailed there. Note that it can take some days for new ltcluster packages to become available via the this repository.

Note

ltcluster RPM packages are designed to be compatible with the community-provided LightDB packages and Hundsun LightDB's 2ndQPostgres. They may not work with vendor-specific packages such as those provided by RedHat for RHEL customers, as the LightDB filesystem layout may be different to the community RPMs. Please contact your support vendor for assistance.

See also FAQ entry Compatability with third party vendor packages.

For more information on the package contents, including details of installation paths and relevant service commands, see the appendix section CentOS packages.

2.2.1.1. Hundsun LightDB public RPM yum repository

Hundsun LightDB provides a dedicated yum public repository for Hundsun LightDB software, including ltcluster. We recommend using this for all future ltcluster releases.

General instructions for using this repository can be found on its homepage. Specific instructions for installing ltcluster follow below.

Installation

  • Locate the repository RPM for your LightDB version from the list at: https://dl.2ndquadrant.com/

  • Install the repository definition for your distribution and LightDB version (this enables the Hundsun LightDB repository as a source of ltcluster packages).

    For example, for LightDB 11 on CentOS, execute: 

    curl https://dl.2ndquadrant.com/default/release/get/11/rpm | sudo bash

    For LightDB 9.6 on CentOS, execute: 

    curl https://dl.2ndquadrant.com/default/release/get/9.6/rpm | sudo bash

    Verify that the repository is installed with: 

    sudo yum repolist

    The output should contain two entries like this: 

    2ndquadrant-dl-default-release-pg11/7/x86_64         Hundsun LightDB packages (PG11) for 7 - x86_64               18
2ndquadrant-dl-default-release-pg11-debug/7/x86_64   Hundsun LightDB packages (PG11) for 7 - x86_64 - Debug        8

  • Install the ltcluster version appropriate for your LightDB version (e.g. ltcluster10): 

    sudo yum install ltcluster11

    Note

    For packages for LightDB 9.6 and earlier, the package name does not contain a period between major and minor version numbers, e.g. ltcluster96.

    Tip

    To determine the names of available packages, execute: 

    yum search ltcluster

Compatibility with PGDG Repositories

The Hundsun LightDB ltcluster yum repository packages use the same definitions and file system layout as the main PGDG repository.

Normally yum will prioritize the repository with the most recent ltcluster version. Once the PGDG repository has been updated, it doesn't matter which repository the packages are installed from.

To ensure the Hundsun LightDB repository is always prioritised, install yum-plugin-priorities and set the repository priorities accordingly.

Installing a specific package version

To install a specific package version, execute yum --showduplicates list for the package in question:

then append the appropriate version number to the package name with a hyphen, e.g.: 

[root@localhost ~]# yum install ltcluster11-4.3-1.el7

2.2.2. Debian/Ubuntu

.deb packages for ltcluster are available from the LightDB Community APT repository (https://apt.postgresql.org/). Instructions can be found in the APT section of the LightDB Wiki (https://wiki.postgresql.org/wiki/Apt).

For more information on the package contents, including details of installation paths and relevant service commands, see the appendix section Debian/Ubuntu packages.

2.2.2.1. Hundsun LightDB public apt repository for Debian/Ubuntu

Hundsun LightDB provides a public apt repository for Hundsun LightDB software, including ltcluster.

General instructions for using this repository can be found on its homepage. Specific instructions for installing ltcluster follow below.

Installation

  • Install the repository definition for your distribution and LightDB version (this enables the Hundsun LightDB repository as a source of ltcluster packages) by executing: 

                  curl https://dl.2ndquadrant.com/default/release/get/deb | sudo bash

    Note

    This will automatically install the following additional packages, if not already present:

    • lsb-release
    • apt-transport-https

  • Install the ltcluster version appropriate for your LightDB version (e.g. ltcluster11): 

                  sudo apt-get install lightdb-11-ltcluster

    Note

    For packages for LightDB 9.6 and earlier, the package name includes a period between major and minor version numbers, e.g. lightdb-9.6-ltcluster.

Installing old packages

See appendix Installing old package versions for details on how to retrieve older package versions.
