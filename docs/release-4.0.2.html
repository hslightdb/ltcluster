A.14. Release 4.0.2

A.14.1. Usability enhancements
A.14.2. Bug fixes
A.14.3. Documentation

Thu Jan 18, 2018

ltcluster 4.0.2 contains some bug fixes and small usability enhancements.

This release can be installed as a simple package upgrade from ltcluster 4.0.1 or 4.0; ltclusterd (if running) should be restarted.

A.14.1. Usability enhancements

  • Recognize the -t/--terse option for ltcluster cluster event to hide the Details column (GitHub #360)

  • Add "--wait-start" option for ltcluster standby register (GitHub #356)

  • Add %p event notification parameter for ltcluster standby switchover

A.14.2. Bug fixes

  • Add missing -W option to getopt_long() invocation (GitHub #350)

  • Automatically create slot name if missing (GitHub #343)

  • Fixes to parsing output of remote ltcluster invocations (GitHub #349)

  • When registering BDR nodes, automatically create missing connection replication set (GitHub #347)

  • Handle missing node record in ltcluster node rejoin (GitHub #358)

A.14.3. Documentation

  • The documentation can now be built as a single HTML file (GitHub pull request #353)
