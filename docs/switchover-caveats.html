7.3. Caveats

  • If using LightDB 21, you should ensure that the shutdown command is configured to use LightDB's fast shutdown mode (the default in 21 and later). If relying on lt_ctl to perform database server operations, you should include -m fast in pg_ctl_options in ltcluster.conf.
  • lt_rewind *requires* that either wal_log_hints is enabled, or that data checksums were enabled when the cluster was initialized. See the lt_rewind documentation for details.
