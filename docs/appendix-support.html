D.1. Reporting Issues

hslightdb provides 24x7 production support for ltcluster and other LightDB products, including configuration assistance, installation verification and training for running a robust replication cluster.

For further details see: https://www.hs.net/lightdb/en/support/

A mailing list/forum is provided via Google groups to discuss contributions or issues: https://groups.google.com/group/ltcluster.

Please report bugs and other issues to: https://github.com/hslightdb/ltcluster.
