C.4.1. Debian/Ubuntu
C.4.2. RHEL/CentOS

C.4.1. Debian/Ubuntu

An archive of old packages (3.3.2 and later) for Debian/Ubuntu-based systems is available here: https://apt-archive.postgresql.org/

C.4.2. RHEL/CentOS

Old versions can be located with e.g.: 

          yum --showduplicates list ltcluster96

(substitute the appropriate package name; see CentOS packages) and installed with: 

          yum install {package_name}-{version}

where {package_name} is the base package name (e.g. ltcluster96) and {version} is the version listed by the yum --showduplicates list ... command, e.g. 4.0.6-1.rhel6.

For example: 

          yum install ltcluster96-4.0.6-1.rhel6

C.4.2.1. ltcluster 3 packages

Old ltcluster 3 RPM packages (3.2 and later) can be retrieved from the (deprecated) Hundsun LightDB repository at http://packages.2ndquadrant.com/ltcluster/yum/ by installing the appropriate repository RPM:

  • http://packages.2ndquadrant.com/ltcluster/yum-repo-rpms/ltcluster-fedora-1.0-1.noarch.rpm
  • http://packages.2ndquadrant.com/ltcluster/yum-repo-rpms/ltcluster-rhel-1.0-1.noarch.rpm
